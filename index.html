GitHub installs a default directory in Documents > Github.
You can use the Github Shell or run > cmd.

Clone a Repository

  1. Open Shell or Terminal
  2. git clone https://github.com/yuanhuynh/github.git
  3. cd directory

Add a File and Upload to Github

  1. git status
  2. git add filename
  3. git commit filename -m "Custom Message Here"
  4. git push

Download files from GitHub

  1. git pull

Github Commands

  1. git add . The period will add all in directory
  2. git add -A The -A will add all in directory
